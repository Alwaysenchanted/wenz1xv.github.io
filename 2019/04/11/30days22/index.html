实现方案

这个span需要满足：

  1. 长宽和目标按钮一致
  2. 位置和目标按钮一致
  3. 随着鼠标移动位置变化
获取按钮属性并赋值

通过一个函数lightOn实现：

通过Element.getBoundingClientRect()返回元素大小及相对视口位置
返回属性(全部只读)：

Attribute Type Description
bottom float Y 轴，相对于视口原点（viewport origin）矩形盒子的底部。
height float 矩形盒子的高度（等同于 bottom 减 top）。
left float X 轴，相对于视口原点（viewport origin）矩形盒子的左侧。
right float X 轴，相对于视口原点（viewport origin）矩形盒子的右侧。
top float Y 轴，相对于视口原点（viewport origin）矩形盒子的顶部。
width float 矩形盒子的宽度（等同于 right 减 left）。
x float X轴横坐标，矩形盒子左边相对于视口原点（viewport origin）的距离。
y float Y轴纵坐标，矩形盒子顶部相对于视口原点（viewport origin）的距离。

这里用到了height、width、left、top分别确定高、宽、x、y值
