#Summary

#####命令
初始化： mysql –initialize –console
安装： mysqld install
启动服务器： net start mysql 关闭： net stop mysql
登陆： mysql -h 主机名 -u 用户名 -p (localhost可以不用-h，初始用户只有root)
修改密码： ALTER USER ‘用户‘@’主机’ IDENTIFIED WITH mysql_native_password BY ‘新密码’
查看数据库： show dadatabases;
